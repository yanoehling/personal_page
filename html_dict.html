Dicionário de Elementos HTML

Voltar para a Página Principal
  • <html>: Define o início do documento HTML
  • <head>: Contém metadados e links do documento
  • <title>: Define o título exibido na aba do navegador
  • <link>: Relaciona arquivos externos como CSS e favicon
  • <body>: Corpo da página, onde o conteúdo é exibido
  • <header>: Cabeçalho da página
  • <nav>: Navegação com links
  • <aside>: Conteúdo lateral, como uma barra lateral
  • <main>: Conteúdo principal da página
  • <section>: Sessões do conteúdo principal
  • <footer>: Rodapé da página
  • <address>: Informações de contato
  • <img>: Imagem
  • <a>: Link para outra página ou recurso
  • <ul> / <li>: Lista não ordenada e seus itens
